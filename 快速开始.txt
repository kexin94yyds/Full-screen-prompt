🎯 这是什么？
--------------
一个可以通过快捷键 Shift+Cmd+O 全局呼出的提示词管理工具。
保留了 Chrome 插件的所有功能，现在可以在 Mac 系统的任何地方使用！


🚀 如何启动？
--------------

方式一：使用启动脚本（推荐）
    1. 打开终端（Spotlight 搜索 "Terminal"）
    2. 拖拽 start.sh 文件到终端窗口
    3. 按回车键

方式二：手动启动
    1. 打开终端
    2. 输入: cd "/Users/apple/mac 提示词库/Slash-Command-Prompter"
    3. 首次运行: npm install
    4. 启动应用: npm start


💡 使用说明
--------------
• 启动后按 Shift + Cmd + O 呼出窗口
• 点击提示词即可复制到剪贴板  
• 支持搜索、分类管理、导入导出
• 所有原插件功能均已保留


📋 主要功能
--------------
✅ 全局快捷键呼出（Shift + Cmd + O）
✅ 多模式分类管理（工作/学习/生活等）
✅ 实时搜索功能
✅ 一键复制到剪贴板
✅ 批量导入导出
✅ 支持上移排序
✅ 现代化界面设计


⚙️ 系统要求
--------------
• macOS 10.13 或更高版本
• Node.js 18+ （如未安装，访问 nodejs.org 下载）

🎨 与 Chrome 插件的区别
--------------
✅ 全局可用 - 不限于浏览器
✅ 快捷键呼出 - Shift+Cmd+O 随时调用
✅ 独立运行 - 不依赖浏览器
✅ 性能更好 - 原生应用响应更快

❌ 暂不支持网页输入框直接插入（但可以复制后粘贴）
