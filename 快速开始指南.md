# Prompter - 快速开始指南

## 🚀 5 分钟快速上手

### 方式一：双击启动（推荐）

```
1. 找到文件：dist/mac-arm64/Prompter.app
2. 双击启动（首次需右键→打开）
3. 按 Shift + ⌘ + O 呼出窗口
```

### 方式二：使用启动脚本

```bash
./启动应用.sh
```

### 方式三：命令行启动

```bash
open "dist/mac-arm64/Prompter.app"
```

---

## ⚡ 核心功能

- **全局快捷键：** `Shift + ⌘ + O`
- **搜索：** 直接输入关键词
- **导航：** ↑ ↓ ← → 方向键
- **选择：** Enter 键（自动复制并粘贴）
- **复制模式：** 点击提示词（仅复制到剪贴板）

---

## ❗ 首次使用必读

### 辅助功能权限设置

**第一次使用"插入"功能时：**

1. 应用会弹出对话框
2. 点击"打开设置"
3. 勾选"Prompter"
4. 完成！无需重启

## 🔄 开发者

### 修改代码后重新打包

```bash
npm run build
```

### 开发模式运行

```bash
npm start
```

⚠️ **注意：** 开发模式需要给终端授予辅助功能权限，建议使用打包后的应用。
